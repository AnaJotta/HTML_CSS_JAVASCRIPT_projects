Proyecto PAC Desarrollo: Web de Conciertos Spice Girls

El proyecto se basa en crear un sitio web de dos páginas (inicio y conciertos) aplicando los conocimientos de HTML5 y CSS3 
siguiendo una estructura de archivos específica.

ESTRUCTURA DEL PROYECTO

- /index.html: Página principal con la descripción del grupo.
- /conciertos.html: Página secundaria con la tabla de fechas de conciertos.
- /README.txt: Este mismo archivo.
- /css/styles.css: Hoja de estilos que da formato a ambas páginas html.
- /img/grupo.jpg: Imagen del grupo usada en la portada realizada con gemini.


CRITERIOS APLICADOS

- Se utiliza HTML5.
- El CSS está enlazado de forma externa y ubicado en la carpeta /css.
- La imagen está ubicada en la carpeta /img.
- Tipografía: Verdana para títulos, Calibri para texto.
- Diseño: fondo claro, secciones en blanco, imagen centrada y de 400px max.
- La tabla de conciertos utiliza colores alternos en las filas.
